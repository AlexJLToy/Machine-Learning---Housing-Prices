# Housing Price Prediction — Machine Learning Exploration

An exploration of classical machine learning techniques applied to the [Kaggle Housing Prices dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques). This project compares multiple regression algorithms to predict residential property sale prices.

---

## 📂 Notebooks

| Notebook | Description |
|---|---|
| `Housing Price Machine Learning - Linear Regression & Regularization.ipynb` | Linear Regression baseline with L1, L2, and Elastic Net regularization |
| `Housing Price Machine Learning - Random Forest.ipynb` | Ensemble Random Forest regressor |
| `Housing Price Machine Learning - Gradient Boosting.ipynb` | Gradient Boosting regressor for improved accuracy |

---

## 🗂️ Dataset

This project uses the Ames Housing dataset from Kaggle, which contains 79 explanatory variables describing nearly every aspect of residential homes in Ames, Iowa.

Download the data here: [House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)

---

## 🛠️ Tech Stack

- **Python 3**
- **Jupyter Notebook**
- **pandas** — data manipulation
- **scikit-learn** — machine learning models
- **matplotlib / seaborn** — visualizations

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AlexJLToy/Machine-Learning---Housing-Prices.git
   cd Machine-Learning---Housing-Prices
   ```

2. Install dependencies:
   ```bash
   pip install pandas scikit-learn matplotlib seaborn jupyter
   ```

3. Download the dataset from Kaggle and place the CSV files in the project directory.

4. Launch Jupyter and open any notebook:
   ```bash
   jupyter notebook
   ```

---

## 📊 Models Covered

- **Linear Regression** — interpretable baseline for regression
- **Regularization (L1 / L2 / Elastic Net)** — techniques to reduce overfitting and handle multicollinearity
- **Random Forest** — bagged ensemble for robust predictions
- **Gradient Boosting** — boosted ensemble for higher predictive accuracy
